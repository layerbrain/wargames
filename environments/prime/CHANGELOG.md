# Changelog

## 0.1.10

- Published the Prime package as `wargames` so `prime env install
  layerbrain/wargames` resolves the public Hub wheel.
- Bundled the WarGames runtime and mission catalog into the Prime package so
  installed environments can load missions outside a source checkout.

## 0.1.9

- Added Prime configs and adapter support for every current WarGames game:
  Doom, SuperTux, Mindustry, Craftium, IKEMEN GO, Open Surge, Quaver, and Naev.
- Updated the Prime README to match the current WarGames catalog and runtime
  contract.

## 0.1.6

- Added game-scoped Prime configs for Red Alert, FlightGear, SuperTuxKart,
  0 A.D., and Freeciv.

## 0.1.0

- Initial WarGames Prime Verifiers environment package.
